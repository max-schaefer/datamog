---
title: "Deductive view of extensional predicates"
kind: content
section: "Pokémon"
---

Extensionals are sets of facts, each a predicate name applied to a tuple of values:

```prolog
pokemon(1, "Bulbasaur", 45)
pokemon(2, "Ivysaur", 60)
…
pokemon(898, "Calyrex", 100)
```

The EDB provides the **initial set of facts** from which we can draw further conclusions.
