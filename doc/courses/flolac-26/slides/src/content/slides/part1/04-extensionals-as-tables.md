Extensional predicates are **tables** consisting of **rows** with the same number of **columns**:

<div class="center">

| id  | name      | hp  |
| --- | --------- | --- |
| 1   | Bulbasaur | 45  |
| 2   | Ivysaur   | 60  |
| …   | …         | …   |
| 898 | Calyrex   | 100 |

</div>

The EDB gives the **contents of extensional predicates** from which we compute other predicates.
