# Pokémon glossary 寶可夢詞彙表

The Part 1 "Datalog Basics" slides use Pokémon as their running example. This
lists every Pokémon, type, move, and ability named on those slides (or carried
in their data), with the official Traditional Chinese (Taiwan) names from the
[Chinese Pokédex](https://tw.portal-pokemon.com/play/pokedex), followed by the
Pokémon-specific vocabulary the slides use.

Like `glossary.md`, this file lives in the slides project root, **outside**
`src/content/slides`, so it is a reference document and not itself a slide.

Since the 2016 official Chinese localisation, these names are the usual way to
refer to Pokémon, types, moves, and abilities in Chinese, though the English
(and Japanese) names are also widely recognised.
